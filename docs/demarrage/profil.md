# Configurer votre profil

Personnalisez vos informations pour une meilleure expérience ElectroHub.

---

## Accéder à votre profil

Il existe deux façons d'accéder à votre profil :

### Depuis le tableau de bord

1. Cliquez sur votre **avatar** (les initiales dans le cercle coloré)
2. La fenêtre de profil s'ouvre

![Clic sur avatar](../assets/screenshots/profile-avatar-click.png)
<!-- Capture d'écran recommandée : L'avatar survolé avec l'icône d'édition -->

### Depuis les cartes d'information

1. Cliquez sur la carte **Site** ou **Département**
2. La fenêtre de profil s'ouvre

---

## Fenêtre de profil

![Fenêtre de profil](../assets/screenshots/profile-modal-full.png)
<!-- Capture d'écran recommandée : La fenêtre modale de profil complète -->

La fenêtre de profil affiche :

### Informations en lecture seule

| Champ | Description |
|-------|-------------|
| **Avatar** | Vos initiales sur fond coloré |
| **Nom** | Votre nom complet |
| **Email** | Votre adresse email professionnelle |

> Ces informations sont gérées par votre administrateur et ne peuvent pas être modifiées directement.

### Informations modifiables

| Champ | Description |
|-------|-------------|
| **Site** | Votre site de rattachement |
| **Département** | Votre service ou département |

---

## Modifier votre site

### Utilisateurs Site

Si vous avez un rôle **Site** :

1. Ouvrez la fenêtre de profil
2. Cliquez sur la liste déroulante **Site**
3. Sélectionnez votre site
4. Cliquez sur **Enregistrer**

![Sélection du site](../assets/screenshots/profile-site-select.png)
<!-- Capture d'écran recommandée : La liste déroulante des sites -->

> **Important** : Vous devez sélectionner un site pour accéder aux équipements.

### Utilisateurs Global/Admin

Si vous avez un rôle **Global** ou **Admin** :

- Vous pouvez changer de site **directement depuis le tableau de bord**
- Utilisez le sélecteur de site dans l'en-tête
- Pas besoin de modifier votre profil à chaque fois

---

## Modifier votre département

Le département aide à organiser les équipes :

1. Ouvrez la fenêtre de profil
2. Cliquez sur la liste déroulante **Département**
3. Sélectionnez votre département
4. Cliquez sur **Enregistrer**

### Départements typiques

- Maintenance
- Production
- Qualité
- Sécurité
- Ingénierie
- Direction

> Les départements disponibles sont configurés par votre administrateur.

---

## Enregistrer les modifications

Après avoir modifié vos informations :

1. Cliquez sur le bouton **Enregistrer** (vert)
2. Attendez la confirmation
3. La fenêtre se ferme automatiquement

![Bouton Enregistrer](../assets/screenshots/profile-save-button.png)
<!-- Capture d'écran recommandée : Le bouton Enregistrer en surbrillance -->

### Messages de confirmation

| Message | Signification |
|---------|---------------|
| ✅ "Profil enregistré !" | Vos modifications sont sauvegardées |
| ❌ "Erreur lors de l'enregistrement" | Un problème est survenu, réessayez |

---

## Annuler les modifications

Pour annuler sans enregistrer :

1. Cliquez sur **Annuler** ou
2. Cliquez sur la croix (**X**) en haut à droite ou
3. Cliquez en dehors de la fenêtre

Les modifications non enregistrées sont perdues.

---

## Impact des paramètres

### Site

Le site sélectionné détermine :

- Les **équipements** que vous voyez
- Les **plans** disponibles
- Les **statistiques** affichées
- L'**historique** accessible

### Département

Le département est utilisé pour :

- **Filtrer** les rapports par équipe
- **Attribuer** les responsabilités
- **Organiser** les notifications

---

## Problèmes courants

### "Aucun site disponible"

Si la liste des sites est vide :
- Votre administrateur n'a pas encore créé de sites
- Contactez votre administrateur

### "Impossible d'enregistrer"

Si l'enregistrement échoue :
1. Vérifiez votre connexion internet
2. Rafraîchissez la page et réessayez
3. Contactez votre administrateur si le problème persiste

### Site non trouvé après enregistrement

Si vous ne voyez pas les équipements de votre site :
1. Déconnectez-vous
2. Reconnectez-vous
3. Vérifiez que le bon site est sélectionné

---

## Bonnes pratiques

### Lors de la première connexion

1. ✅ Configurez immédiatement votre site
2. ✅ Sélectionnez votre département
3. ✅ Vérifiez que les données s'affichent correctement

### En cas de changement d'affectation

Si vous changez de site :
1. Mettez à jour votre profil
2. Informez votre administrateur pour ajuster les droits si nécessaire

### Pour les utilisateurs multi-sites

- Utilisez le sélecteur de site sur le tableau de bord
- Évitez de modifier le profil à chaque changement
