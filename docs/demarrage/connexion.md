# Connexion à l'application

Cette page vous guide pour vous connecter à ElectroHub.

---

## Accéder à ElectroHub

### URL de connexion

Ouvrez votre navigateur web et accédez à l'adresse fournie par votre administrateur :

```
https://votre-entreprise.electrohub.app
```

> **Note** : L'URL exacte dépend de la configuration de votre entreprise. Contactez votre administrateur si vous ne la connaissez pas.

### Page de connexion

![Page de connexion](../assets/screenshots/signin-page.png)
<!-- Capture d'écran recommandée : La page de connexion avec les champs email/mot de passe -->

La page de connexion présente :
- Le logo ElectroHub
- Un champ pour votre **adresse email**
- Un champ pour votre **mot de passe**
- Un bouton **Se connecter**
- Un lien **Mot de passe oublié**

---

## Se connecter

### Étape 1 : Saisir vos identifiants

1. Entrez votre **adresse email** professionnelle
2. Entrez votre **mot de passe**

### Étape 2 : Cliquer sur "Se connecter"

Cliquez sur le bouton **Se connecter** pour accéder à l'application.

### Étape 3 : Accéder au tableau de bord

Si vos identifiants sont corrects, vous êtes redirigé vers le **tableau de bord**.

![Tableau de bord après connexion](../assets/screenshots/dashboard-welcome.png)

## Première connexion

Lors de votre première connexion, vous devrez peut-être :

### 1. Configurer votre profil

Un message vous invite à compléter votre profil :

- **Site** : Sélectionnez votre site de rattachement
- **Département** : Choisissez votre service

> Voir [Configurer votre profil](./profil.md) pour plus de détails.

### 2. Découvrir l'interface

Prenez quelques minutes pour explorer :
- Les sections du tableau de bord
- La barre de navigation
- Les applications disponibles

---

## Mot de passe oublié

Si vous avez oublié votre mot de passe :

### Étape 1 : Cliquer sur "Mot de passe oublié"

Sur la page de connexion, cliquez sur le lien **Mot de passe oublié**.

### Étape 2 : Saisir votre email

Entrez l'adresse email associée à votre compte.

### Étape 3 : Vérifier votre boîte mail

Un email contenant un lien de réinitialisation vous sera envoyé.

### Étape 4 : Créer un nouveau mot de passe

Cliquez sur le lien dans l'email et choisissez un nouveau mot de passe.

> **Attention** : Le lien de réinitialisation expire après quelques heures. Si vous ne l'utilisez pas à temps, recommencez la procédure.

## Problèmes de connexion

### "Email ou mot de passe incorrect"

Vérifiez que :
- Votre adresse email est correctement saisie
- Votre mot de passe est correct (attention aux majuscules)
- Le verrouillage des majuscules (Caps Lock) n'est pas activé

### "Compte non trouvé"

Votre compte n'existe peut-être pas encore. Contactez votre administrateur pour :
- Vérifier que votre compte a été créé
- Obtenir vos identifiants de connexion

### "Session expirée"

Pour des raisons de sécurité, votre session expire après une période d'inactivité. Reconnectez-vous simplement.

### Écran blanc ou erreur

Si l'application ne charge pas :
1. Rafraîchissez la page (F5 ou Ctrl+R)
2. Videz le cache de votre navigateur
3. Essayez un autre navigateur
4. Contactez votre administrateur

## Déconnexion

Pour vous déconnecter :

1. Cliquez sur votre **avatar** ou **nom** dans la barre de navigation
2. Sélectionnez **Déconnexion**

![Menu de déconnexion](../assets/screenshots/logout-menu.png)
<!-- Capture d'écran recommandée : Le menu utilisateur avec l'option de déconnexion -->

> **Bonne pratique** : Déconnectez-vous toujours lorsque vous quittez un poste partagé.

---

## Sécurité

### Bonnes pratiques

- **Ne partagez jamais** votre mot de passe
- Utilisez un mot de passe **unique et complexe**
- **Déconnectez-vous** sur les postes partagés
- Signalez immédiatement tout accès suspect

### Authentification sécurisée

ElectroHub utilise :
- Connexion HTTPS chiffrée
- Tokens JWT sécurisés
- Expiration automatique des sessions
- Protection contre les attaques courantes
