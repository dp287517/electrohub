**ElectroHub** est une application web professionnelle dédiée à la **gestion de la maintenance électrique industrielle**. Conçue pour les équipes de maintenance, elle centralise tous les aspects de la gestion des équipements électriques, des contrôles périodiques et de la conformité réglementaire.

## Pourquoi ElectroHub ?

Dans un environnement industriel, la gestion de la maintenance électrique est cruciale :

- **Sécurité** : Les équipements électriques mal entretenus représentent un risque majeur
- **Conformité** : Les réglementations (ATEX, contrôles périodiques) exigent une traçabilité complète
- **Efficacité** : Centraliser les informations réduit les temps d'intervention
- **Planification** : Anticiper les maintenances évite les arrêts de production

ElectroHub répond à ces enjeux en offrant une solution complète et intuitive.

## Fonctionnalités principales

### Gestion des équipements

| Module | Description |
|--------|-------------|
| **Tableaux électriques** | Modélisation par bâtiment/étage/local, gestion des appareillages |
| **Haute Tension (HT)** | Cellules HT, câbles, transformateurs, jeux de barres |
| **Variateurs (VSD)** | Variateurs de fréquence, puissances, contrôles périodiques |
| **Équipements mécaniques** | Pompes, ventilateurs, moteurs |
| **Équipements globaux** | Onduleurs, batteries de compensation, éclairage de sécurité |
| **Équipements mobiles** | Perceuses, meuleuses et outillage électrique |

### Sécurité et conformité

| Module | Description |
|--------|-------------|
| **ATEX** | Gestion des atmosphères explosives et équipements certifiés |
| **Portes coupe-feu** | Contrôles annuels, QR codes, non-conformités |
| **Arc Flash** | Calcul de l'énergie incidente et catégories EPI |
| **Sélectivité** | Coordination des protections |
| **Courant de défaut** | Études de court-circuit |

### Outils avancés

| Outil | Description |
|-------|-------------|
| **Cartographie** | Visualisation des équipements sur plans |
| **Assistant IA** | Aide à la planification et recherche intelligente |
| **Export PDF/Excel** | Génération de rapports professionnels |
| **Historique d'audit** | Traçabilité complète des modifications |

## Pour qui ?

ElectroHub s'adresse aux :

- **Techniciens de maintenance** : Accès rapide aux informations terrain
- **Responsables maintenance** : Pilotage et planification des interventions
- **Ingénieurs électriciens** : Études techniques et analyses
- **Responsables sécurité** : Suivi de la conformité réglementaire
- **Prestataires externes** : Accès contrôlé aux équipements
