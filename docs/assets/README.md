# Assets pour la documentation GitBook

Ce dossier contient les ressources visuelles de la documentation.

## Structure

```
assets/
├── screenshots/    # Captures d'écran de l'application
├── icons/          # Icônes et badges
└── images/         # Autres images (schémas, logos)
```

## Recommandations

### Dimensions

- Largeur recommandée : 1200-1600 px
- Format : PNG (préféré) ou JPG
- Taille max : 500 Ko par image

### Contenu

- Utilisez des données de démonstration
- Masquez les informations sensibles
- Assurez une bonne lisibilité

### Nommage

Utilisez des noms en minuscules avec tirets :
- `module-name-description.png`
- Pas d'espaces ni de caractères spéciaux
