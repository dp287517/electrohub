Le module **Formation ATEX** (Learn Ex) propose un parcours de formation au niveau 0 pour les atmosphères explosives.

---

## Présentation

![Vue Formation ATEX](../assets/screenshots/learnex-overview.png)

## Module 1 : Introduction

### Objectifs

- Comprendre ce qu'est une atmosphère explosive
- Connaître les enjeux de sécurité
- Identifier les réglementations applicables

### Contenu

#### Qu'est-ce qu'une ATEX ?

Une **atmosphère explosive** est un mélange d'air avec des substances inflammables (gaz, vapeurs, brouillards, poussières) susceptible de s'enflammer.

#### Triangle du feu

```
         Comburant
            /\
           /  \
          /    \
         /______\
   Combustible   Énergie
                d'activation
```

Les trois éléments doivent être présents pour qu'une explosion se produise.

#### Réglementation

- **Directive 2014/34/UE** (ATEX 95) : Équipements
- **Directive 99/92/CE** (ATEX 137) : Protection des travailleurs

## Module 2 : Les risques

### Objectifs

- Identifier les sources de combustible
- Reconnaître les sources d'inflammation
- Évaluer les risques

### Contenu

#### Sources de combustible

| Type | Exemples |
|------|----------|
| **Gaz** | Méthane, propane, hydrogène |
| **Vapeurs** | Essence, solvants, alcools |
| **Brouillards** | Huiles pulvérisées |
| **Poussières** | Farine, sucre, aluminium |

#### Sources d'inflammation

1. Surfaces chaudes
2. Flammes et gaz chauds
3. Étincelles mécaniques
4. Étincelles électriques
5. Électricité statique
6. Foudre
7. Ondes électromagnétiques
8. Ultrasons
9. Compression adiabatique
10. Réactions chimiques

## Module 4 : Les équipements

### Objectifs

- Lire un marquage ATEX
- Comprendre les modes de protection
- Vérifier la conformité

### Contenu

#### Marquage ATEX

Exemple : `⟨Ex⟩ II 2 G Ex d IIB T4 Gb`

| Élément | Signification |
|---------|---------------|
| ⟨Ex⟩ | Marquage CE ATEX |
| II | Groupe (industries de surface) |
| 2 | Catégorie |
| G | Gaz |
| Ex d | Mode de protection |
| IIB | Groupe de gaz |
| T4 | Classe de température |
| Gb | Niveau de protection |

#### Quiz interactif

Des exercices pratiques pour apprendre à décoder les marquages.

## Module 5 : Comportement

### Objectifs

- Connaître les règles de comportement en zone
- Savoir réagir en cas d'incident
- Appliquer les consignes

### Contenu

#### Règles de base

✅ **À faire** :
- Vérifier l'autorisation d'accès
- Porter les EPI requis
- Utiliser des outils conformes
- Signaler toute anomalie

❌ **À ne pas faire** :
- Fumer
- Utiliser un téléphone non ATEX
- Créer des étincelles
- Modifier les installations

#### En cas d'urgence

1. Alerter (alarme, responsable)
2. Évacuer la zone
3. Ne pas déclencher d'étincelles
4. Attendre les secours

## Module 6 : Quiz final

### Évaluation

20 questions à choix multiples couvrant tous les modules.

### Seuil de réussite

**80%** de bonnes réponses (16/20) pour valider la formation.

### En cas d'échec

- Possibilité de repasser le quiz
- Révision des modules recommandée
- Pas de limite de tentatives

---

## Attestation

### Génération

Après réussite du quiz :
1. Attestation générée automatiquement
2. Téléchargeable en PDF
3. Enregistrée dans votre profil

### Contenu de l'attestation

- Nom du participant
- Date de formation
- Score obtenu
- Durée de validité (généralement 3 ans)
- Signature électronique

### Validité

La formation niveau 0 est généralement valide **3 ans**. Un recyclage est ensuite nécessaire.

## FAQ

### Cette formation remplace-t-elle une formation présentielle ?

Le niveau 0 peut être fait en e-learning. Les niveaux supérieurs nécessitent généralement du présentiel.

### Puis-je interrompre et reprendre plus tard ?

Oui, votre progression est sauvegardée.

### Comment prouver ma formation ?

Téléchargez l'attestation PDF depuis votre profil.

### La formation est-elle obligatoire ?

Pour le personnel intervenant en zone ATEX, oui. Pour les autres, c'est une sensibilisation recommandée.
